# tiktok
## 一、项目介绍
一句话介绍，实现极简版抖音。每一个应用都是从基础版本逐渐发展迭代过来的，希望同学们能通过实现一个极简版的抖音，来切实实践课程中学到的知识点，如 Go 语言编程，常用框架、数据库、对象存储等内容，同时对开发工作有更多的深入了解与认识，长远讲能对大家的个人技术成长或视野有启发。  
项目服务地址-目前为本地服务  
Github  地址：https://github.com/Aserck-ck/tiktok
